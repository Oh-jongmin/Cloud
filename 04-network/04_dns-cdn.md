# 🌎 DNS & CDN

---

## ✅ DNS (Domain Name System)

- 도메인 이름을 IP로 변환
- AWS의 DNS 서비스: **Route 53**

## ✅ Route 53 기능

- 도메인 등록, 헬스 체크, 지능형 라우팅 (Latency, Failover 등)

## ✅ CDN (Content Delivery Network)

- 전 세계 엣지 로케이션에 콘텐츠 캐싱
- 빠른 응답 시간, 대역폭 절감

> ✅ AWS: CloudFront / Azure: Front Door / GCP: Cloud CDN
