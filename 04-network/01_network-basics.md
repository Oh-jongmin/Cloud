# 🌐 네트워크 기본 개념 (Network Basics)

---

## ✅ IP 주소 & CIDR

- **IP 주소**: 장치 고유 식별자 (예: 192.168.0.1)
- **CIDR 표기법**: IP 범위 지정 방식 (예: /24 → 256개 IP)

## ✅ 퍼블릭 vs 프라이빗 네트워크

| 구분 | 퍼블릭 | 프라이빗 |
|------|--------|-----------|
| 접근성 | 인터넷에서 접근 가능 | 내부 전용, 보안 강화 |
| 예 | 웹 서버 | DB 서버, 백엔드 앱 |
