# 📦 Amazon EBS (Elastic Block Store)

---

## ✅ 개요

- EC2 인스턴스에 붙이는 디스크 볼륨
- SSD / HDD 유형 존재

## ✅ 유형

| 타입 | 설명 | 용도 |
|------|------|------|
| gp3 | 범용 SSD | 대부분의 워크로드 |
| io2 | 프로비저닝 IOPS | DB, 고속처리 |
| sc1 | Cold HDD | 드물게 접근하는 데이터 |

## ✅ 스냅샷

- S3에 저장되는 백업
- 볼륨 복구, 복제에 사용

> ⚠️ 인스턴스를 종료해도 볼륨은 별도 과금됨
